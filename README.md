# Interview Scheduler

## Setup

Install dependencies with `npm install`.

## Running Webpack Development Server

```sh
npm start
```

## Running Jest Test Framework

```sh
npm test
```

## Running Storybook Visual Testbed

```sh
npm run storybook
```

## Dependencies

- Axios
- Classnames
- Normalize CSS
- React
- React-DOM
- React-Scripts
- Babel
- Storybook
- Cypress
- Node-Sass
- Prop-types
- React-test-renderer
- @testing-library/react
